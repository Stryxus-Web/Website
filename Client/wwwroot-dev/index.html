Major Browser Tests

WebAssembly

Some browsers have WebAssembly disabled, most commonly on organisational devices over fears of security.

WebAssembly is the core of what runs this website allowing the .NET Runtime to be run in your browser and C# code to be executed

AVIF Decoding

AVIF is used for this websites imagery and animations due to its extreme effeciency when compared to legacy formats like JPEG and PNG. It also supports a plethora of features like HDR formatting capabilities. AVIF is based off of AV1 but implemented seperatly.

WebM/AV1 Decoding

AV1 is used for this websites videos due to its extreme effeciency when compared to that of h264, h265 and more including capabilities like HDR formatting.

FLAC Decoding

FLAC is used for this website and is a royalty free audio format designed for hi-res audio at about half the size of formats like WAV.

CSS: Backdrop Filter

Backdrop Filter is used on many devices natively these days and the web has had support for it for a few years now but its not all there still, and it makes for a much more modern experience.

Minor Browser Tests

135 PPI Screen

This website is designed for high PPI screens for sharp crisp detail.

PPI (Pixels per inch) is calculated by using Pythagorean Theorem using the screens pixels in the form of √ width2 x height2 to get the diagonal length in pixels and then dividing that against the physical diagonal length in inches.

WebP Decoding

WebP is an image format designed by Google to succeed the legacy JPEG, PNG and GIF formats.

WebM/VP9 Decoding

WebM is a video/audio container designed by Google which supports VP9 to compete with that of h264 and is the relative of WebP in many respect while also supporting AV1.
